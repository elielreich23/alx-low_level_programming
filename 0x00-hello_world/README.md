<p dir="auto">To run the Betty linter just with command <code>betty &lt;filename&gt;</code>:</p>
<ul dir="auto">
<li>Go to the <a href="https://alx-intranet.hbtn.io/rltoken/wQ4sMfsWfxvyfN67Sc11zA" title="Betty" rel="nofollow">Betty</a> repository</li>
<li>Clone the <a href="https://alx-intranet.hbtn.io/rltoken/wQ4sMfsWfxvyfN67Sc11zA" title="repo" rel="nofollow">repo</a> to your local machine</li>
<li><code>cd</code> into the Betty directory</li>
<li>Install the linter with <code>sudo ./install.sh</code></li>
<li><code>emacs</code> or <code>vi</code> a new file called <code>betty</code>, and copy the script below:</li>
</ul>
